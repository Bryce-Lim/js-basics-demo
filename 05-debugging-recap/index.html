Debugging & Recap

Open DevTools → Console. Look for red errors. Use console.log to inspect values.

Task
  1. Click console.log demo and watch the console.
  2. Enable Introduce a bug, then click Run buggy code.
  3. Read the red error, click the filename to jump to the line.
Tip

Breakpoints: DevTools → Sources → Click line number to pause execution.
