JavaScript Follow‑Along — 00 Starter

Goal: set up a page, link a JS file, print to the console.

Instructions

  1. Open index.html in VSCode.
  2. Create a file named script.js.
  3. Make sure the <script src="script.js"> tag appears before </body>.
  4. Open the browser DevTools console and refresh.

You should see: Hello from script.js
